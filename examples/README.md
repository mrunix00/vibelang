# VibeLang Examples

This directory contains example programs written in VibeLang.

## Examples

- `hello.vibe`: Prints a simple message.
- `fib.vibe`: Computes the 10th Fibonacci number.
- `factorial.vibe`: Computes factorial of 5.
- `calculator.vibe`: Performs basic arithmetic operations.
- `loop.vibe`: Demonstrates a while loop summing numbers.
- `strings.vibe`: Shows string concatenation.
