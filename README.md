# Vibelang

A programming language interpreter written 100% by AI.

## Build instructions

### Clone the repo:

```console
git clone --recursive https://github.com/mrunix00/vibelang.git
```

### Build the project:

```console
cd vibelang\
make
```

### Run the tests (optional):

```console
make test
```

### Run the interpreter:

```console
./build/main <file.vibe>
```
