# Syntax

## Declarations

```js
let x = 5; // Declare an integer variable `x` with value 5
let y; // Declare a variable `y` with value null
let z = "Something"; // Declare a string variable `z` with value "Something"
let w = true; // Declare a boolean variable `w` with value true
let pi = 3.1415; // Declare a float variable `pi` with value 3.1415
```

## Assignment

```js
x = 10; // Assign value 10 to variable x
y = "Hello"; // Assign string "Hello" to variable y
```

## Expressions

```js
let x = 5 + 2; // x = 7
let y = x - 3; // y = 4
let z = x * 3; // z = 21
let w = y + " World"; // w = "4 World"
```

## Control Flow

```js
if (x > 0) {
  print("x is positive");
} else {
  print("x is not positive");
}
```

## Loops

```js
while (x > 0) {
  print(x);
  x = x - 1;
}
```

## Functions

```js
function add(a, b) {
  return a + b;
}

let result = add(3, 4); // result = 7
```
